VARIABLES
Son espacios de memoria a los que les asignaré un dato para utilizarlo más adelante.

STRINGS
Son cadenas de caracteres alfanumericos, pueden ser letras, números o símbolos.

FUNCIONES
Son grupos de sentencias a las que por cuestiones de comodidad, les asignamos un nombre y que luego utilizaremos varias veces. Se crean con el fin de ahorrarnos escribir código.
A estas funciones les podemos "pasar" argumentos, es decir, datos que serán utilizados por estas para retornarnos algún valor. Por ejemplo a la funcion suma, le pasamos como argumentos dos numeros, los que serán sumados dentro de la función y luego esta nos devolverá el resultado utilizando 'return'. 'return' se utiliza para que la función devuelva un valor.

IF
La declaración 'if' se utiliza como condicional, es decir, cuando se cumple una condición, ejecuta una sentencia, sino no. También podemos hacer que el 'if' evalúe otras condiciones si no se cumple la primera (utilizando 'else if'), o que ejecute otra sentencia.

VALORES BOOLEANOS
Valores booleanos son true y false. Son el resultado de hacer comparaciones (un numero es mayor que otro, un numero es menor que otro, una variable es igual a otra, una variable es igual o menor a 5,etc). Si la comparación es verdadera, da como resultado 'true', si es falsa, devuelve 'false'. Utilizamos los siguientes operadores booleanos:
< menor que
> mayor que
<= menor o igual a
>= mayor o igual a
!== diferente de
=== igual a

Ejemplos de expresiones booleanas:
4>3
da como resultado 'true', ya que 4 es mayor que 3

5<1
devuelve 'false', 5 es mayor que 1

'argentina'==='Argentina'
las cadenas no son idénticas, devolverá 'false'

5>=5
devuelve 'true' ya que 5 no es mayor que 5, pero sí es igual.
